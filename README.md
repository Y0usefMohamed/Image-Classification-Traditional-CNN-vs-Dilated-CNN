1. Introduction
Deep learning has become one of the most important technologies in computer vision and image processing. Convolutional Neural Networks (CNNs) are widely used for image classification because they can automatically learn important visual features from images.
This project focuses on comparing two different CNN approaches:
•	Standard Convolution
•	Dilated Convolution
The comparison is performed using two famous datasets:
•	CIFAR-10
•	STL-10
The purpose of the project is to understand how changing the convolution type affects image classification performance and feature extraction.
